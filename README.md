# Safe RAG Benchmark

Benchmark for evaluating safety of retrieval-augmented generation systems.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m safe_rag_benchmark
```

## Project structure

```
safe_rag_benchmark/        # source package
tests/      # tests
requirements.txt
```
